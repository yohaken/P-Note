# ชื่อแถบ (ใช้เรียกตอนคุย)

ตัดคำว่า「ทั้งหมด」ออก — เรียกสั้นๆ ตามนี้:

| ชื่อเรียก | ความหมาย | ชิปในแถบ |
|-----------|----------|----------|
| **กำหนดเวลา** | จัดเรียงรายการ | ล่าสุด · ตามกำหนด · อิสระ |
| **ความสำคัญ** | กรองระดับความสำคัญ | สำคัญ+ด่วน · สำคัญ · เร่งด่วน · ทั่วไป |
| **แท็ก** | กรองตามแท็ก | ตามแท็กที่สร้าง |
| **กลุ่มงาน** | สถานะงาน (เมนู ☰) | งาน · ทำแล้ว · ถังขยะ |
| **สุขภาพ** | ปุ่มใน Bottom Navigation (UI เตรียมโหมดสุขภาพ) | `#health-mode-btn` |
| **เพิ่ม** | สร้างโน้ตใหม่ (ใน Bottom Navigation) | `#add-note-btn` |

ตัวอย่าง: 「ปรับแถบความสำคัญ」「ย้ายกำหนดเวลาไปด้านบน」「แก้กลุ่มงาน」「ปุ่มสุขภาพ」
